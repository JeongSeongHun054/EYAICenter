# K-Means Clustering for User Segmentation
from sklearn.cluster import KMeans
import pandas as pd

# Load processed logs
df = pd.read_sql("SELECT * FROM process_logs", conn)
X = df[['session_count', 'duration_minutes']]

# Run K-Means with optimal k=3
kmeans = KMeans(n_clusters=3, random_state=42)
df['cluster'] = kmeans.fit_predict(X)

print("클러스터별 통계:")
print(df.groupby('cluster').mean())

회계감사 Audit — AI 기반 이상 거래 탐지 및 감사 위험 평가

전통적인 샘플링 감사 기법을 넘어, 전사 트랜잭션 데이터를 머신러닝으로 지능적으로 분류하고 이상 항목을 정밀하게 추출하여 감사의 정확도와 커버리지를 동시에 높입니다.

  • 이상 거래 탐지: 분산 원장 로그를 정규화하여 회계 기준과 다른 변칙 거래 패턴을 Random Forest 분류기로 자동 식별
  • 문서 자동 파싱: 계약서 및 영수증 이미지에서 GPT Vision API로 금액, 날짜, 거래처를 추출하여 전표 정합성 자동 검수
  • 감사 위험 예측: 기업 재무 데이터를 분류 모델에 입력하여 고위험 과목과 감사 우선순위를 사전에 식별
  • 시간 절감 효과: 기존 수작업 대비 이상 항목 검출 속도 약 8배 향상, 감사인 재검토 집중 효율화

세무자문 Tax — 세액 공제 최적화 및 세무 리스크 사전 식별

법인세 신고 데이터와 정책 데이터베이스를 LLM으로 교차 분석하여 미적용 공제 항목을 자동으로 탐지하고 세무 리스크 항목을 선제적으로 플래그합니다.

  • 공제 항목 자동 탐지: 연구개발 세액공제, 고용증대 세액공제, 통합투자세액공제 누락 여부를 데이터 기반으로 검출
  • 세무 리스크 스크리닝: 접대비 한도 초과, 감가상각 일관성, 이월결손금 미활용 등 리스크 항목 자동 분류
  • 시뮬레이션 리포트: 공제 적용 시 예상 세액 절감액을 시나리오별로 계산하여 의사결정 자료 자동 생성
  • LLM 자연어 해설: 복잡한 세법 조문을 GPT API로 클라이언트 맞춤 언어로 번역하여 커뮤니케이션 효율화

전략컨설팅 Parthenon — 물류 네트워크 최적화 및 운영 효율화
