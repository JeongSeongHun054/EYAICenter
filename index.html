📋 AI Center 핵심 요구자격 매칭
1. AI / Gen AI / Data Science 프로젝트

파생 변수를 반영한 케이민즈 군집화 및 랜덤포레스트 예측 알고리즘을 설계하여 유저 성향과 경기 잔류 확률을 머신러닝으로 도출하는 데이터 사이언스 모듈을 구현했습니다.

🔍 실증 분석 결과표 확인
2. 데이터 전처리 및 Analytics 툴 활용

3개년 대량의 설비 기동 트랜잭션 로그를 SQLite 및 Pandas로 정제하고, 기기별 상태 변동성에 지수평활법과 Z-score 통계를 적용해 공정 버퍼 임계값과 이송 트리거를 연산하는 파이프라인을 구축했습니다.

🔍 임계값 정합성 실증 데이터 확인
3. Physical AI 원천기술 [Isaac Sim / Omniverse]

엔비디아 Isaac Sim의 자율 이동 로봇 물류 개념을 2D 환경으로 단순화해 구현했습니다. 다익스트라 최단 경로 연산 및 이송 상태에 따른 실시간 텔레메트리 위치 좌표 스트림을 우측 시뮬레이터 탭에서 구동하고 있습니다.

🔍 3D 기반 시뮬레이터 실행
4. LLM API 활용 및 프롬프트 테스트

OpenAI GPT API 연동을 가정한 대화형 인터랙티브 위젯을 우측 비즈니스 탭에 설계했습니다. 감사, 세무, 전략 탭을 누른 상태에서 쿼리를 입력하면 각 업무 컨텍스트에 특화된 프롬프트 분석 답변을 실시간으로 확인해 볼 수 있습니다.

🔍 GPT API 프롬프트 플레이그라운드
5. 서비스라인 AI 비즈니스 지원

EY의 주요 본부별 AI 컨설팅 시나리오를 수립했습니다. 감사 본부의 분산 원장 이상 거래 자동 탐지, 세무 본부의 공제 대상 분석, 파르테논 본부의 물류 경로 단축 및 재배치 이관 알고리즘을 상세 Use Case로 설계했습니다.

🔍 서비스라인 유스케이스 열람
6. 생성형 AI / 머신러닝 / 통계분석 활용

포트폴리오 하단 탭을 통해 파이썬 스마트팩토리 재배치 이송 최적화 알고리즘, 씨플플 다익스트라 로봇 경로 탐색 핵심 로직, 그리고 데이터웨어하우스 적재 검증을 위한 SQL 쿼리를 직접 조회하고 분석할 수 있는 체계를 마련했습니다.

회계감사 [Audit] AI 활용 및 데이터 검증 지원

전통적인 샘플링 감사 기법을 넘어, 전사적 트랜잭션 데이터를 지능적으로 검증하고 이상 항목을 추출하여 감사의 정밀도를 높입니다.

  • 이상 거래 탐지: 분산 원장 로그 데이터를 정규화하여 회계 기준과 다른 변칙 거래 패턴을 머신러닝으로 분석
  • 문서 자동 파싱: 계약서 및 영수증 이미지 자료에서 키 가치 정보와 일자, 수량을 추출하여 정합성 검수 자동화
  • 감사 위험 예측: 기업 규모와 과거 재무 상태 데이터를 머신러닝 분류 모델에 대입하여 감사 위험 단계 선제 식별
