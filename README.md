# EY한영 AI Center 인턴 지원 포트폴리오
## 제조 물류 스마트팩토리 디지털 트윈 및 Physical AI 실증 플랫폼

본 프로젝트는 엔비디아 Isaac Sim의 실물 가상화 개념을 웹 기반으로 설계한 물류 이송 시뮬레이터와 비즈니스 분석용 생성형 AI 플레이그라운드를 탑재한 통합 포트폴리오 웹 서비스입니다.

---

### 핵심 요구자격 부합도 및 기술 증명

구분 | 채용공고 자격 요건 | 본 프로젝트 포트폴리오 내 구현 및 실증 내용
--------------------------------------------------------------------------------------
요건 1 | AI 및 Data Science 프로젝트 | 파생 변수 피처 엔지니어링 기반 케이민즈 유저 성향 분류 및 랜덤포레스트 게임 이송 성공 판별 예측 모델 설계
요건 2 | 데이터 전처리 및 분석 툴 활용 | 3개년 누계 9.9만 건의 공정 기동 트랜잭션 데이터를 SQLite 및 Pandas로 정제하고 기기 상태 변동성을 고려해 공정 임계값 도출
요건 3 | Physical AI 원천기술 활용 | 최단 경로 다익스트라 알고리즘과 2D 캔버스를 연동하여 로봇 실시간 이동 상태 및 텔레메트리 좌표 스트림 시각화
요건 4 | LLM API 활용 및 프롬프트 테스트 | OpenAI GPT API 연동을 모사한 프롬프트 테스트 베드를 가동하여 업무별 특화 컨텍스트 프롬프팅 답변 생성
요건 5 | 서비스라인 AI 비즈니스 지원 | EY의 주요 본부별 컨설팅 유스케이스 수립하여 감사 본부, 세무 본부, 파르테논 본부 맞춤 분석 기능 탑재
요건 6 | 머신러닝 및 통계분석 활용 | 웹 화면에서 파이썬 최적화 코드와 씨플플 알고리즘 소스 및 데이터 정합성 검증용 SQL 쿼리를 직접 조회하는 코드 뷰어 연동

---

### 주요 아키텍처 및 구현 기술

* **프론트엔드**: 웹 브라우저 내에서 지연 없이 즉각 실행되는 순수 HTML5 및 CSS3와 바닐라 자바스크립트 기반 싱글 페이지 대시보드
* **시뮬레이션 엔진**: 400 곱하기 240 규격 캔버스 렌더링 루프 기반의 실시간 2D 로봇 제어 시스템
* **알고리즘**: 가중치 기반 최단 거리를 탐색하는 다익스트라 경로 탐색 알고리즘

---

### 디렉토리 구조

* **index.html**: 대시보드 화면 마크업 및 요구자격 매칭 현황판 배치 파일
* **styles.css**: 92vh 높이 제한의 카드 섀도우 다크 테마 디자인 및 모바일 반응형 그리드 스타일 시트
* **app.js**: 로봇 위치 연산 루프 및 OpenAI 모델 응답 딜레이 모사 함수와 탭 화면 제어 로직을 관리하는 자바스크립트 파일
